3장 데이터구조 : 여러 데이터를 한 변수에 담는 목적
               
     2절. 튜플 : (리스트와 유사하나 수정 불가)
         하나의 요소만 가질 때는 , 포함 ex) tup=(1,)
         ,가 없으면 스칼라 데이터로 바뀐다.
     3절. 딕셔너리 : {키와 값의 쌍}
          dic.keys(): 키 목록 /
          dic.values() : 값 목록
          dic.items() : 키-값 목록
          읽기 : 변수[키] 보다는 get(키)를 사용하여 에러를 줄이자
          추가/수정 : 변수[키] = 값 / 변수.update(추가하거나 수정할 값)
          삭제 : pop(zl) / del 변수[키]
     4절. 셋 : {중복을 허용하지 않는 집합}. 순서(index)가 없음
          추가 : add(스칼라 데이터), update(데이터구조)
          삭제 : remove(데이터)
     5절. enumerate
          인덱스나 가상의 순번을 추가한 순환 객체 : for문 사용
