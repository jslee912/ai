주피터 노트북 닫은 상태에서 아나콘다 프롬프트에서 아래 6개 명령실행

conda install -c conda-forge jupyter_nbextensions_configurator             =>확장 설정 UI 설치 (conda 방식)

pip install jupyter_nbextensions_configurator jupyter_contrib_nbextensions  =>확장 기능 및 설정 UI 설치 (pip 방식)
jupyter contrib nbextension install --user                                               =>확장 기능들을 사용자 계정에 등록
jupyter nbextensions_configurator enable --user                                     =>설정 UI를 사용자 환경에서 활성화

jupyter contrib nbextension install --sys-prefix                                        =>가상환경 내에 확장 기능 등록
conda install nbclassic=0.4.8                                                                =>옛날 Notebook UI 유지 (확장 호환성용)
